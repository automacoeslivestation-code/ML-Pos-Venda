# Produtos — Cameras de Seguranca e Acessorios

## Cameras IP / WiFi
- Suportam acesso remoto via app (preencher com apps suportados)
- Resolucoes disponiveis: 1080p, 2MP, 4MP, 5MP, 8MP (4K)
- Visao noturna infravermelha: alcance de 20m a 80m dependendo do modelo
- Armazenamento: cartao SD (ate 256GB) e/ou nuvem

## Cameras Analogicas / HDCVI / AHD
- Requerem DVR compativel com o padrao (HDCVI, AHD, TVI, CVBS)
- Cabo coaxial RG59 recomendado para distancias acima de 30m

## DVRs e NVRs
- DVR: para cameras analogicas
- NVR: para cameras IP (PoE ou WiFi)
- Canais disponiveis: 4, 8, 16, 32

## Acessorios
- Fontes: 12V para cameras individuais, fontes chaveadas para multiplas cameras
- Cabos: RG59 (analogico), cabo de rede Cat5e/Cat6 (IP)
- Suportes e brackets: incluso na maioria dos produtos
